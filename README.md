# Events
This is an events scheduler backend api

This application is used to learn but not limited to;
- Clean architecture
- Dependency injection
- Dependency injection container
- Email handling
- Repository
- Generics
- Interfaces
- Entity Framework Core
- SOLID principles
- Defensive Coding
- Sql Server database
- Azure Blob storage
